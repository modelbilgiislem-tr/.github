# Security Policy

Model Bilgi İşlem olarak güvenlik bildirimlerini ciddiyetle ele alıyoruz.

Eğer public repository içeriklerimizde, örnek projelerimizde, dokümantasyonlarımızda veya geliştirici kaynaklarımızda güvenlik açısından risk oluşturabilecek bir durum fark ederseniz, lütfen bunu public issue olarak paylaşmayın.
