# Support

Model Bilgi İşlem GitHub alanı; geliştirici kaynakları, örnek projeler, dokümantasyonlar ve teknik içerikler için kullanılmaktadır.

Kurumsal destek, ürün desteği, müşteri talepleri ve canlı sistem problemleri için GitHub issue alanı yerine resmi iletişim kanallarımızı kullanmanızı rica ederiz.

## Resmi Destek Kanalları

Website: https://www.mbi.com.tr/  
E-posta: info@mbi.com.tr  
Telefon: 0 212 275 87 88

## GitHub Üzerinden Destek

GitHub issue alanı yalnızca aşağıdaki konular için kullanılmalıdır:

- Public repository içeriğiyle ilgili hata bildirimi
- Dokümantasyon eksikliği
- Örnek proje sorunları
- Geliştirici kaynaklarıyla ilgili öneriler
- Açık kaynak kapsamındaki teknik geri bildirimler

## GitHub Üzerinden Paylaşılmaması Gerekenler

Lütfen GitHub üzerinde aşağıdaki bilgileri paylaşmayın:

- Müşteri bilgileri
- Kişisel veriler
- Lisans bilgileri
- Canlı sistem erişim bilgileri
- Veritabanı bağlantı bilgileri
- API anahtarları
- Şifreler
- Ticari veya gizli operasyon bilgileri

## Yanıt Süreci

GitHub üzerinden açılan issue ve talepler, public repository kapsamına göre değerlendirilir.

Kurumsal müşteri destek süreçleri için resmi destek kanallarımız önceliklidir.
